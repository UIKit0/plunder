Plunder

tl = new Timeline(mySquare)
tl.forever ->
  group = tl.together duration: 2000, ->
    tl.rotate
      from: 720
      to: 0
    tl.color
      from: [255, 0, 0, 1]
      to: [0, 0, 255, 0]
    tl.scale
      from: 1
      to: 10
    tl.move
      from: x: 10, y: 10
      to: x: 300, y: 100
  tl.wait 500
  tl.reverse(group)

Plunder is an animation system targeted at JavaScript game engines. It provides powerful tween based animations, is engine agnostic, and easily integrates into your game.

Plunder is written in CoffeeScript, well tested and performant. It is in its infancy right now, but was extracted from a previous game engine I wrote, so will mature quickly. The goal is to make it easily compatible with all major game engines, like ImpactJS, FrozenJS, MelonJS and more.
